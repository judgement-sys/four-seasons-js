# four-seasons-js
A small educational website created using HTML, CSS and JavaScript.

## Features

- Selecting a season using buttons.
- Changing the title, text, background color and image.
- Four seasons:
  - Spring
  - Summer
  - Autumn
  - Winter

## Project Structure

index.html
style.css
script.js
images/

## Technologies

- HTML5
- CSS3
- JavaScript
